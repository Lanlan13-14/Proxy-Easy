### 安装
```
curl -L https://raw.githubusercontent.com/Lanlan13-14/Proxy-Easy/refs/heads/main/proxy.sh -o /tmp/proxy-easy && chmod +x /tmp/proxy-easy && sudo mv /tmp/proxy-easy /usr/local/bin/proxy-easy && proxy-easy
```
### 2. 已安装？执行
```
sudo proxy-easy
```
